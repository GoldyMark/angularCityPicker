ngCityPicker Demo

直辖市：

北京、天津、上海、重庆是直辖市，选择省份后直接跳转到县区选择框；


不设市辖区的地级市：

目前包括四个不设市辖区的地级市：广东中山、广东东莞、甘肃嘉峪关、海南三亚，这些城市不设县区，选择城市后直接关闭选择框；

HTML代码：

<mk-city-picker open="params.open" placeholder="请选择城市" format="$0省-$1市-$2县" ng-model="params.outputModel" q="params.inputModel">


JS代码：

              angular.module('myApp', ['ngCityPicker'])
              .controller('myCtrl', ['$scope', function($scope){
                  $scope.params = {
                      outputModel:null,
                      open:false,
                      // inputModel:"440106",
                      inputModel:['广东','广州','天河']
                  }
              }]);
